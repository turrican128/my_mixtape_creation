# Cover art base image

Drop **any image** here (jpg, jpeg, png, webp, bmp) and the app will use it
as the background for your Mixcloud cover art.

- If multiple images are present, the first one (alphabetical order) wins.
- To pin a specific image when you have several, rename it to
  `cover_base.jpg`, `cover_base.jpeg`, or `cover_base.png` — those preferred
  names always take priority.

The mixtape title is overlaid on top using one of the built-in presets
(Neon, Chrome, Outrun). The image is automatically square-cropped and
resized to 1500×1500.

For best results:

- Use a **square-ish** image (anything else will be center-cropped).
- Aim for **at least 1500×1500 pixels** so the output is not upscaled.
- High-contrast backgrounds work best — the text overlay sits in the lower
  third with a glow/shadow to help legibility.

If no image is present here, the upload still works, but no cover art will
be attached to the Mixcloud upload.
